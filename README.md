 # 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built using **Tableau** and **Microsoft Excel** to analyze employee attrition and workforce insights.

## 🔗 Live Dashboard

## 🛠️ Tools Used
- Tableau
- Microsoft Excel

## 📈 Dashboard Highlights
- Employee Attrition Analysis
- Workforce Demographics
- Department-wise Insights
- Job Role Analysis
- Salary & Performance Insights

## 🎯 Purpose
This dashboard helps HR professionals understand employee trends, identify factors affecting attrition, and make data-driven workforce decisions.
